Tag Guide

How Tags Work

Every article gets one content-type tag and then a few topic tags. The first tag tells you the format. The rest tell you the subject. Notes is the default for informative posts that are not a review or project.

Content Type Tags

Build / Show

Project Demo

Project is for something I built or engineered. Demo is for a prototype, example, or interactive showcase.

Explain / Teach

Notes Review

Notes covers documentation-style or explanatory posts. Review is for an opinionated take after using something.

Challenge / Exam

Writeup Cert

Writeup is reserved for HTB, CTF, and challenge walkthroughs. Cert is for certification-focused posts and exam experiences.

Lookup / Utility

Reference

Reference is for cheatsheets, lookup pages, and content meant to be revisited quickly.

Topic Tags

Topic tags are the technologies, platforms, or domains attached to the post. Common examples include Python, Maps, Security, HTB, and API.
