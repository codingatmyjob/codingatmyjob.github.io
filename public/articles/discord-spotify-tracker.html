Overview

This is going to be a brief code review. First, I think a little bit of backstory is required. This was one of the first serious Python projects I built as a live service; it served a community of thousands of users, it enabled a livestream where outages actually mattered, and it was maintained entirely solo. Here's what happened.

What feels like a lifetime ago I was incredibly passionate for music production. One line of effort I pursued was working as a producer, and managing the discord of a newly formed label. This role I functioned a plethora of skills, but I'm going to focus on the code aspect. I coded an auto-role system, setup the entire discord from scratch, created custom chat filtering (things needed to be removed specific to the Twitch stream), and coded a custom Spotify activity tracker. There was no pay, there were no perks. I poured hundreds, if not thousands of hours into this project, and when I resigned the owner didn't care, until I turned off my discord bot.

That may sound dramatic, but the Discord bot I made ran nearly half the label's Twitch stream, and the Spotify Activity Tracker was the most valuable feature. During streams, it posted an embed of the current song every 60 seconds, reflecting the activity of a particular user. This way, users receive live updates on the song being played and reviewed on the Twitch stream, and can interact with these embeds. This was the core feature enabling the side by side Twitch/Discord integration, for users who wanted to listen on Twitch, and chat/make requests in the discord. I'm going to try and do a modern analysis of the Spotify code block, everything I did wrong, and how I would do it now.

The Original Code

Out of respect, I've removed any information that could identify the Discord, users, or the label I was working with from the code snippet below. Everything else is exactly how I left it in August of 2023. 

########################### SONG COMMAND #############################
@bot.command(name="song", aliases=["nowplaying", "current", "np"])
@commands.has_any_role("Admin", "*", "Content Team")
async def now_playing(ctx, user: discord.Member = None):
    if user == None:
        user = user
        pass
    while spotify.enabled == True:
        if user.activities:
            for activity in user.activities:
                if isinstance(activity, Spotify):
                    channel = bot.get_channel(123456789012345678) # channel id
                    embed = discord.Embed(title = "**NOW PLAYING:**", description = "Listening to {}".format(activity.title), color = 0xa22b20)
                    embed.set_thumbnail(url=activity.album_cover_url)
                    embed.add_field(name="Artist", value=activity.artist)
                    embed.add_field(name="Track Link", value=f"[{activity.title}](https://open.spotify.com/track/{activity.track_id})")
                    embed.set_footer(text="Song started at {}".format(activity.created_at.strftime("%H:%M")))
                    await channel.send(embed=embed)
                    print ("Shared current music activity.")
                    await asyncio.sleep(60)
                else:
                    skip
        else:
            await asyncio.sleep(60)
            print ("Bot needs music activity.")
            continue
    else:
        skip
